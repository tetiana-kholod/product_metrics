# product_metrics
This project contains an analysis of user behavior data. The original dataset was in Excel format, followed by data cleaning and preparation. An interactive dashboard was built in Tableau, featuring product metrics, dual‑axis charts, a retention funnel, cohort analysis.

### Skills Showcased

This project was a journey through key Excel and Tableau features.
**Data Transformation** was performed in Excel. Cleaned, shaped, and prepared the raw data for analysis by duplicates removal, changing data types, creating new columns, Pivot Tables, using XLOOKUP, etc.

Here’s a look at what is mastered in Tableau:
- **KPI Indicators & Tables:** Using **Cards** to display key product metrics and **Tables** for cohort analysis.
-	**Core charts:** Used **Dual-axis charts (Bar and Line/ Gantt Bar), funnel chart**
-	**Dashboard Design:**  Designed an intuitive and visually appealing layout, exploring different to best tell the data story.
-	**Filter** to dynamically filter the report by date.
-	**Show/Hide Button** for Filter.
